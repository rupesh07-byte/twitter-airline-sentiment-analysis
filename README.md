# ✈️ Twitter Airline Sentiment Analysis Dashboard

## 📊 Project Overview

This project analyzes customer sentiment towards major US airlines using Twitter data.
The workflow includes **data cleaning, analysis, and visualization using Python**, followed by building an **interactive dashboard in Power BI** to extract business insights.

---

## ⚙️ Project Workflow

1. Data Collection (Twitter Airline Dataset)
2. Data Cleaning & Preprocessing (Python - Pandas)
3. Exploratory Data Analysis (Python - Seaborn & Matplotlib)
4. Dashboard Development (Power BI)
5. Insight Generation

---

## 📁 Dataset

* Source: Twitter US Airline Sentiment Dataset
* Total Records: 14,640 tweets
* Features: Airline, Sentiment, Negative Reasons, Date, Text

---

## 🛠 Tools & Technologies

* Python (Data Cleaning & Analysis)
* Pandas (Data Processing)
* Seaborn & Matplotlib (Data Visualization)
* Power BI (Dashboard & Visualization)

---

## 📌 Key Insights

* ⚠️ 62.7% of tweets are negative
* Major issues: Customer Service & Flight Delays
* United Airlines has the highest negative sentiment
* Sentiment trends fluctuate over time
